Engenharia de software

O curso de Engenharia de Software tem como principal objetivo, formar profissionais capacitados para suprir a demanda do mercado de trabalho da área de TI - Tecnologia da Informação no país.


Diferencial

A disciplina Banco de Dados Não Relacional (NoSQL), é uma disciplina muito atual e moderna, onde são abordados temas como desenvolvimento e manipulação de dados não estruturados. Sendo o sistema de gerenciamento de banco de dados utilizado em Big Data. O segundo destaque para esse curso é a disciplina Computação em nuvem, também uma disciplina que reflete a infraestrutura que as aplicações são disponibilizadas atualmente. Devido a convergência dos serviços para a nuvem, o profissional com tais conhecimentos é extremamente necessário para prover as operações nas empresas. Como terceiro destaque, e não menos importante que os demais, existe uma disciplina chamada Segurança em Engenharia de Software. Muitas IES oferecem a disciplina segurança da informação, porém, esse curso precisa de conhecimento em segurança, mas totalmente voltado ao desenvolvimento de sistema propriamente dito. Vale a pena ressaltar que, essa indicação partiu do time de produção, e ainda verificado que nenhuma das IES analisadas oferecem tal disciplina.
